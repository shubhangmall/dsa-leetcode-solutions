# [2235. Add Two Integers](https://leetcode.com/problems/add-two-integers/)

## Problem Description
Given two integers `num1` and `num2`, return the **sum** of the two integers.

**Example:**
- **Input:** `num1 = 12, num2 = 5`
- **Output:** `17`
- **Explanation:** `num1 + num2 = 12 + 5 = 17`, so 17 is returned.

## Intuition
This is the most fundamental problem on LeetCode. It tests the basic ability to use the addition operator (`+`) in your chosen programming language. Since the problem guarantees the input will be within the range of a standard 32-bit integer, we can simply return the direct sum of the two variables.

## Complexity
- **Time Complexity:** $O(1)$, as addition is a constant time operation for fixed-width integers.
- **Space Complexity:** $O(1)$, as we do not use any additional data structures or memory.
